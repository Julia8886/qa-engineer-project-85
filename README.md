Тестирование платформы Codebattle
О проекте Codebattle — это интерактивная игровая платформа для соревновательного программирования.Концепция: два разработчика одновременно решают одну и ту же алгоритмическую задачу на скорость, используя разные языки программирования.Цель тестирования: проверка корректности работы компиляторов, игровой механики (определение победителя в реальном времени), синхронизации комнат через WebSocket и стабильности веб-интерфейса под нагрузкой.

Содержимое репозитория 
В этом репозитории хранятся артефакты ручного тестирования и документация проекта:
docs/checklists/ — Чек-листы для проверки функционала (регистрация, создание комнат, игровой процесс, профиль игрока)
docs/test-cases/ — Тест-кейсы для сложных сценариев (обрыв соединения во время боя, одновременная отправка верного кода)
docs/matrices/ — Матрица покрытия поддерживаемых языков программирования и корректности работы компилятора
fixtures/ — Примеры кода (валидные решения и синтаксические ошибки) для ручной вставки в редактор при проверках
README.md — Описание проекта и структуры артефактов.
